# FORGE — Умный дневник тренировок

Персональный фитнес-трекер с автоподбором программы, картой восстановления мышц, личными рекордами и видео техники упражнений.

## Возможности

- **41 упражнение** с обучающими видео
- **Автоподбор тренировки** на основе восстановления мышц и доступного инвентаря
- **Карта восстановления** мышц (heatmap)
- **Личные рекорды** с автоматическим детектом
- **Прогрессивная перегрузка** — приложение помнит последние веса
- **Замена упражнения** одним тапом
- **Заметки и самочувствие** к каждой тренировке
- **PWA** — устанавливается на телефон как приложение
- **Офлайн-работа** — благодаря Service Worker
- **Защита данных** — Storage Persistence API + бэкапы (экспорт/импорт)

## Хранение данных

Данные хранятся локально в браузере (`localStorage`). Это означает:
- Никаких серверов и аккаунтов
- Полная приватность
- **НО** — данные могут быть удалены при очистке кеша браузера

### Защита от потери данных

В приложении встроено несколько уровней защиты:

1. **Storage Persistence API** — приложение запрашивает у браузера разрешение
   не удалять данные автоматически. Удалить можно только вручную в настройках.

2. **Бэкапы JSON** — кнопки "Скачать" и "Загрузить" в настройках. Файл можно
   сохранить в облако (iCloud/Google Drive) или на компьютер.

3. **Автонапоминания** — каждые 10 тренировок или раз в 30 дней приложение
   предлагает сделать бэкап.

4. **Видимый счётчик** в настройках показывает дату последнего бэкапа.

### Перенос на новое устройство

1. На старом устройстве: Настройки → "Скачать бэкап"
2. На новом устройстве: открой приложение → Настройки → "Загрузить"
3. Можно объединить с уже существующими данными или заменить полностью.

## Деплой на Cloudflare Pages

### Вариант A: Через GitHub (рекомендуется)

Этот способ даёт автодеплой при каждом `git push`.

#### 1. Создай репозиторий на GitHub

```bash
# В папке проекта
git init
git add .
git commit -m "Initial commit"

# Создай репо на github.com (через UI или gh CLI)
gh repo create forge --public --source=. --push

# Или вручную: создай репо, затем
git remote add origin https://github.com/USERNAME/forge.git
git branch -M main
git push -u origin main
```

#### 2. Подключи репо к Cloudflare Pages

1. Зайди на [dash.cloudflare.com](https://dash.cloudflare.com)
2. Зарегистрируйся (бесплатно) если ещё нет аккаунта
3. В левом меню: **Workers & Pages** → **Create** → **Pages** → **Connect to Git**
4. Выбери GitHub, разреши доступ к репозиторию `forge`
5. На экране настройки сборки:
   - **Project name**: `forge` (это станет частью домена: `forge.pages.dev`)
   - **Production branch**: `main`
   - **Framework preset**: `None`
   - **Build command**: оставь пустым
   - **Build output directory**: `/` (корень)
6. Жми **Save and Deploy**

Через ~1 минуту приложение будет доступно по адресу:
**`https://forge.pages.dev`**
(если имя `forge` занято — Cloudflare предложит вариант с суффиксом)

Каждый последующий `git push` в `main` автоматически деплоится.

---

### Вариант B: Через Wrangler CLI (без GitHub)

Подходит если не хочешь возиться с git.

#### 1. Установи Wrangler

```bash
npm install
```

#### 2. Авторизация

```bash
npx wrangler login
```

Откроется браузер — войди в Cloudflare-аккаунт.

#### 3. Деплой

```bash
npm run deploy
```

При первом запуске Wrangler спросит имя проекта — введи `forge`. Готово, приложение задеплоено.

Для последующих обновлений просто запускай `npm run deploy`.

## Локальная разработка

### Простой вариант (любой статический сервер)

```bash
# Если установлен Python
python3 -m http.server 8000

# Или через Node
npx http-server . -p 8000 -c-1

# Или через npm script
npm run preview
```

Открой `http://localhost:8000`.

### С эмуляцией Cloudflare (для тестирования _headers и _redirects)

```bash
npm run dev
```

Открой `http://localhost:8788`. Здесь работают `_headers`, `_redirects` и переменные окружения как в проде.

## Установка как приложение (PWA)

После деплоя приложение можно установить на телефон/компьютер:

**На iPhone (Safari):**
1. Открой приложение в Safari
2. Нажми "Поделиться" (квадрат со стрелкой вверх)
3. Выбери "На экран Домой"

**На Android (Chrome):**
1. Открой приложение в Chrome
2. В меню (⋮) выбери "Установить приложение" или "Добавить на главный экран"

**На десктопе (Chrome/Edge):**
1. В адресной строке справа появится иконка установки (⊕)
2. Кликни и подтверди установку

## Кастомизация после деплоя

### Свой домен

1. На странице проекта в Cloudflare Pages: **Custom domains** → **Set up a custom domain**
2. Введи свой домен (например `forge.example.com`)
3. Если домен уже на Cloudflare — настройка автоматическая
4. Если на другом регистраторе — Cloudflare подскажет какие DNS-записи добавить

### Изменение названия и метаданных

- **Название приложения**: `manifest.json` (поля `name`, `short_name`)
- **Описание и SEO**: метатеги в `index.html`
- **Цвета темы**: CSS-переменные в `css/styles.css` (секция `:root`)
- **Логотип**: пересобрать иконки (есть Python-скрипт в исходниках) или заменить вручную
